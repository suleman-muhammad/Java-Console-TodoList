# Java Console ToDo List

## Overview
A lightweight task management application built with Java. This project demonstrates core Object-Oriented Programming (OOP) concepts including inheritance, interfaces, and file I/O for data persistence.

## Features
- Add, Edit, and Delete tasks.
- **Data Persistence:** Saves tasks to a .txt file so data isn't lost on close.
- Exception handling for invalid user inputs.

## Technologies Used
- Java 17
- OOP Principles (Polymorphism, Encapsulation)

## How to Run
1. Clone the repository.
2. Compile: `javac Main.java`
3. Run: `java Main`
